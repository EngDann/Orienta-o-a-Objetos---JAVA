1. São a forma mais básica de dados em Java
    byte: 8-bit, inteiro com sinal.
    short: 16-bit, inteiro com sinal.
    int: 32-bit, inteiro com sinal.
    long: 64-bit, inteiro com sinal.
    float: ponto flutuante de 32-bit.
    double: ponto flutuante de 64-bit.
    char: 16-bit, Unicode character.
    boolean: true ou false.

2. É o processo de converter explicitamente um tipo de dados em outro.
    - Convertendo um tipo menor para um tipo maior.
        int myInt = 9;
        double myDouble = myInt; 

    -  Convertendo um tipo maior para um tipo menor.
        double myDouble = 9.78;
        int myInt = (int) myDouble;
